# Recruitment — Experiment 001

Target: 3–5 independent participants (not the maintainer).

---

## What to send

Share only this URL:

**https://jon4ohio.github.io/anchor-lab/**

Do not send Anchor README, Journey, or orient-project playbook.

---

## Facilitator checklist

- [ ] Participant has not read Anchor documentation beforehand (best effort)
- [ ] No coaching during the lesson
- [ ] Note question numbers where participant hesitates
- [ ] After completion, optional post-test (3 unseen items — facilitator provides verbally)
- [ ] Log session in [research/experiment-001-observations.md](../research/experiment-001-observations.md)
- [ ] Optional: ask if they will try adopting Anchor on their own project (observe only)

---

## Stop condition

If 2 consecutive participants abandon before Q3, pause recruitment and revise lesson-v0 before continuing.

---

## After 3–5 sessions

Write [research/experiment-001-synthesis.md](../research/experiment-001-synthesis.md).

Only then consider citing findings in the Anchor repository.
